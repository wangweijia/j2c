# SubBridge CN

一个完全免费、面向 Netflix 的 Chrome 扩展 MVP：将日语/英语字幕实时翻译为中文，并在屏幕底部展示。

## 已实现（初始化版本）

- Manifest V3 扩展骨架
- Netflix 页面字幕监听（多选择器策略）
- 日语/英语基础语言检测
- 免费翻译管线（在线免费接口 + 本地回退）
- 中文覆盖层渲染（底部）
- Popup 快速开关
- Options 样式设置
- 快捷键切换：Option+Shift+T（macOS）
- 字幕延迟微调（-500ms ~ +500ms）
- 拼句防抖与自动清屏
- 翻译状态徽标（翻译中/缓存命中/实时翻译）

## 快速启动

1. 打开 Chrome 扩展管理页：chrome://extensions
2. 打开开发者模式
3. 点击“加载已解压的扩展程序”
4. 选择当前项目根目录 j2c
5. 进入 Netflix 任意播放页验证

## 注意事项

- 免费在线翻译接口存在速率限制，已提供本地回退策略。
- 本项目不绕过 DRM，不抓取视频，仅处理页面字幕文本。
